# Ruby CLI Gem - The Bath & Beauty Finder
    
This gem allows the user to search through and choose various products for skin and body. Users can choose from a vast inventory of shower gels, body creams, body lotions, and fragrance mists.

# Getting Started

You'll need the following ruby gems to be able to run this application:

gem 'nokogiri (1.11.1-x86_64-darwin)'
gem 'pry (0.13.1)'
gem 'open-uri (0.1.0)'
gem 'pastel (0.8.0)'
gem 'colorize'

You can find the updated version of each gem on "https://rubygems.org"

To clone this repo, you can find the repo on https://github.com/nanaama527/beauty-finder/

# Installing

In order to install each gem and have it run, create a "config" folder, which holds an "environment" file. Require all the gems by calling each with the command:

    "require '____' "

To ensure that all gems will run on all files, make sure to include the command:

    "require_relative (../lib/____)"

In the terminal, install each gem by using the command, "gem install". When finished, be sure to run the command by running "bundle gem install" in the terminal.
